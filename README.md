# Owais J's Blog

## Tech Stack

- SvelteKit
- TailwindCSS

## Features

- Two column with a sidebar
- Post carousel
- Responsive
- Swappable reading font
- Dark mode
- Search
- Tags
- Pagination
- RSS feed
- Sitemap
- Github as a CMS

## Design System

### Typography

- Heading: `Josefin Sans`
- Text: `Lato`
- Mono: `Overpass Mono`
